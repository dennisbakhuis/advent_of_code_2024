## Install
Requirements:
- [Conda (I use micromamba)](https://mamba.readthedocs.io/en/latest/user_guide/micromamba.html)
- [Poetry](https://python-poetry.org/)

Then:
```bash
conda env create -f environment.yml
conda activate aoc2024

poetry install
```
